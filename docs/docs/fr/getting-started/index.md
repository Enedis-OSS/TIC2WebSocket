# Démarrage

Cette section explique comment démarrer rapidement avec TIC2WebSocket.

## Étapes rapides 
1. Vérifiez les prérequis matériels et logiciels. [Prérequis](prerequisites.md)
2. Installez TIC2WebSocket en suivant les instructions d'installation. [Installation](installation.md)
3. Lancez le serveur TIC2WebSocket. [Lancement](launch.md)
4. Utilisez le testeur WebSocket pour envoyer des requêtes et recevoir des données TIC. [Testeur WebSocket](ws-tester.md)
