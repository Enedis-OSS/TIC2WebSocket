# Getting Started

This section explains how to get started quickly with TIC2WebSocket.

## Quick steps

1. Check the required hardware and software. [Prerequisites](prerequisites.md)
2. Install TIC2WebSocket by following the installation instructions. [Installation](installation.md)
3. Start the TIC2WebSocket server. [Launch](launch.md)
4. Use the WebSocket tester to send requests and receive TIC data. [WebSocket Tester](ws-tester.md)
